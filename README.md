# PulsePay Landing Page

This repo contains a static `index.html` landing page for **PulsePay** (Tailwind via CDN + Iconify icons).

## Run locally (Windows)

Node.js is the easiest way to preview.

### Option A: built-in Node server (no downloads)

```powershell
cd C:\Users\USER\Downloads\pulsepay
node server.js
```

Then open `http://localhost:5173`.

### Option B: `serve` via npx

```powershell
cd C:\Users\USER\Downloads\pulsepay
npx --yes serve@latest . --listen 3000
```

Then open `http://localhost:3000`.

## Files

- `index.html`: the full website (styles + scripts included).
